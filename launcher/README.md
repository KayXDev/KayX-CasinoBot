# 🎰 Casino Bot Launcher - Simple Version

## 🚀 Uso Rápido

**Ejecuta el launcher:**
```
launcher\CasinoBot-Launcher.bat
```

## 🎯 Características

✅ **Launcher simple y limpio**  
✅ **Interfaz bonita con colores**  
✅ **Solo lo esencial**  
✅ **Fácil de usar**

## 📋 Opciones Disponibles

| Opción | Función |
|--------|---------|
| [1] Start Bot | Inicia el bot |
| [2] Stop Bot | Para el bot |
| [3] Restart Bot | Reinicia el bot |
| [4] Status Check | Ver estado del sistema |
| [5] View Config | Ver configuración |
| [6] Quick Test | Prueba rápida del sistema |

## ⚡ Requisitos

- Windows 10/11
- Node.js instalado
- MySQL server
- Bot configurado en config.yml
